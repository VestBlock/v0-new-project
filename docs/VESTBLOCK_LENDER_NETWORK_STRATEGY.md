# VestBlock Lender Network Strategy

VestBlock's lender network is designed as a structured partner graph, not a flat spreadsheet.

## Goals

- Build nationwide lender coverage across real estate, business, personal, and specialty programs.
- Match borrower profiles to lenders with better fit logic before outreach or referral.
- Track lender relationships like a lightweight CRM inside VestBlock.
- Grow partner coverage by market, product, and borrower segment over time.

## Initial seeding priority

Start with the lenders VestBlock can monetize and route fastest:

1. DSCR / bridge / fix-and-flip lenders
2. Community banks and credit unions
3. SBA and startup-friendly business lenders
4. CDFIs and local economic-development programs
5. Bilingual / Spanish-friendly lenders

## Coverage model

Each lender should eventually have:

- product/program tags
- state coverage
- contact path
- startup / investor / owner-occupied tolerance
- credit / revenue / time-in-business minimums
- referral notes
- outreach status
- performance history

## Compliance stance

- VestBlock does not imply lender approval.
- Partner outreach stays in review mode unless explicitly approved.
- Public-source discovery only. No scraping of private dashboards or prohibited sources.
- Referral language should stay truthful and documentable.
