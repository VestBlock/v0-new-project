## Weekly Proof Rhythm

- Monday: Capture proof screenshots with `npm run visibility:proof-assets`.
- Tuesday: Publish one founder post using the latest proof score and one buyer-friendly lesson.
- Wednesday: Submit VestBlock to two relevant directories or partner pages.
- Thursday: Publish or prepare one short video with links back to the exact service page.
- Friday: Update the scorecard, note what changed, and add the best proof to the case study.
