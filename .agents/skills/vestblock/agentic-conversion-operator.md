# Agentic Conversion Operator

Use this skill when making VestBlock easier for humans and AI agents to complete high-value tasks.

## Inputs

- public landing pages
- request forms
- thank-you states
- preview flows
- dashboard entry points
- partner redirect routes

## Goal

Make core VestBlock flows more completion-friendly for:

- small-business visitors
- real-estate borrowers
- sellers
- lenders / buyers / partners
- AI browsing agents and structured task completion

## Priority flows

1. request AI receptionist setup
2. request visibility plan
3. submit business funding details
4. submit real-estate funding deal
5. submit seller property details
6. open partner / lender handoff routes

## Focus

- remove blank-state friction
- make form fields explicit and machine-legible
- preserve important deal context through submit, thank-you, alerts, and dashboard
- expose “what happens next” clearly
- improve agent discoverability of form purpose, action, and required fields

## Deliverables

- completion-flow audit
- task-friction map
- field / copy fixes
- thank-you / post-submit continuity fixes
- partner-handoff context preservation

## Guardrails

- do not leak internal ops wording into public UX
- do not rely on hidden assumptions for required deal details
- keep task flows understandable without a sales call
- prefer durable field names and obvious confirmation states
