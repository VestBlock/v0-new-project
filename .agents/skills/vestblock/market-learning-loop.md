# Market Learning Loop

Use this skill when improving city rotation, niche rotation, and market re-queue decisions.

## Learn from

- lead volume
- send volume
- reply rate
- bounce rate
- booked or won outcomes
- market score trends

## Safe auto-updates

- re-queue strong cities sooner
- keep winning niches in rotation
- deprioritize weak markets gradually

## Review-required changes

- major state expansion shifts
- aggressive revisit timing changes
- drastic threshold changes
