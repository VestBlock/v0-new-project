# Admin Intelligence Operator

Use this skill when building or refining operator-facing dashboards for VestBlock.

## Surfaces

- `/admin-panel`
- `/admin/improvement`
- `/admin/research`
- `/admin/experiments`

## Priorities

- surface what matters now
- reduce digging
- separate low-risk auto-improvements from queued approvals
- make daily wins, losses, and actions visible fast

## Good operator outcomes

- admins know what worked yesterday
- admins know what is stuck
- admins can approve useful changes quickly
- admins can trace why the system recommended a change
