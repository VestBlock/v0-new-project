# Website Weakness Audit

Use this skill when classifying a business lead as a website upgrade or automation opportunity.

## Check for

- website missing
- mobile weakness
- no obvious CTA
- no chat widget
- no booking path
- weak contact signals
- no trust signals
- old/thin content

## Routing

- weak trust + weak CTA -> `Website Upgrade Sprint`
- no chat + missed-contact signal -> `AI Receptionist Launch`
- no booking + service business -> `AI Appointment Booking System`

## Notes

- keep findings practical
- phrase them as opportunity signals, not insults
- store the audit in `website_audit_json`
